김규빈

1. 인형 관련 사이트
요즘 인형놀이 하는 사람 많으니까 인형 놀이에 쓰기 좋은 아이템들을 모아서 추천해줌
커뮤니티 형식
뭐 다이소 미니어처 제품 같은 거... 모아서 가격이나 용도나 생김새같은 거 나열하고...
사람들이 좋아보이는 제품들 막 올리면 관리자가 확인해서 카테고리별로 올려줌 

2. 꽃다발 사이트 
싄이가 말해준 거... 얘도 판매 비슷함 
꽃다발을 팜... 꽃말이나 꽃관련이벤트날 알려줌 뭐 성인식엔 장미라던가 그런거... 

3. 옷입혀주는 쇼핑몰
엄마가 말해준 거... 사용자의 전신 사진에 각종 쇼핑몰의 옷을 입혀보는 느낌... 근데 얘는 좀 애매하고 어려운 거 같음 웹보다는 앱의 느낌? 

4. 정모 사이트
그냥 말그대로 정모할 지리?를 추천해줌
3명이서 정모한다 치면 그 셋의 주소에서 가장 거리가 비슷한... 삼각의 중심을... 추천함 그리고 주변 추천 먹거리나 장소나 축제 그런 거 있으면 좋을 거 같음 지도앱에 응용하면 재밌지않을까

2. 전철 실시간 정보 사이트
- 참고 사이트 : 지하철 api 활용, 서울교통공사 실시간 열차 운행 정보 사이트
- 단점 : 티맵교통정보 어플처럼 유사한 어플 너무 많음

3. 여행 사이트
- 참고 사이트 : 스카이스캐너
- 코로나로 인한 입국 가능국가, 입국해서 준비할 서류 등 확인할 수 있게 도와주는 사이트

4. 혼밥 커뮤니티
- 참고사이트 : 망고플레이트
- 위치 기반으로 혼밥하기 좋은 식당을 알려줌 + 맛 평점, 혼밥난이도 등

5. 품앗이
- 참고 사이트 : 알바몬, 당근마켓
- 동네 사람들에 한해 노동력을 제공할 수 있도록 하는 사이트
- 단점 : 검증된 신원정보인지 확인할 수 있는 방법 없음
